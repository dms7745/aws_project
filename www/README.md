# AWS 기반의 클라우드 네이티브 애플리케이션 설계 및 배포
본 사이트는 개별 운영 중인 LMS 프로젝트와 AI 비식별화 서비스를 통합하여 관리하고, 프로젝트별 산출물을 정리한 웹 기반 포트폴리오입니다.

## URL https://www.da2un.store

## 1. 프로젝트 개요
- 개발 기간: 2025.12 ~ 2026.01
- 주요 목적:
  분산된 개별 서비스(LMS, AI 분석)의 통합 진입점 구축
  Nginx 리버스 프록시를 활용한 다중 도메인 및 서버 운영 환경 실습
- 활용 방안: 실질적인 서비스 운영 / 인프라 자동화 / 보안 강화 다양한 기술 스택으로 이루어진 서비스의 통합 운영

## 2. 주요 기능
- 서비스 라우팅: 각 프로젝트(LMS, AI) 도메인으로의 연결 및 통합 인터페이스 제공
- 분석 요청 관리: 사용자가 업로드한 영상의 처리 상태 확인 및 결과물 서빙
- 운영 관리: Nginx를 활용한 포트 분기 및 SSL 인증서 적용을 통한 안정적인 웹 서비스 제공
